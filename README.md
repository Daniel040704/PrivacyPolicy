# PrivacyPolicy
Last updated: December 12, 2025

This Privacy Policy explains how Courier Sallary (“we”) collect, use, and protect user data in the Courier Sallary application (“App”).
1. Information We Collect

1.1 Personal Information
When using the App, we collect:
email address
password (securely stored using Firebase Authentication)

1.2 Account Management
We use Firebase Authentication to manage user accounts, login, and password recovery.

1.3 Salary Calculation Data
Users may enter salary-related data.
This data:
is stored in Firebase Firestore / Realtime Database or locally on the device
is used only for app functionality
is not shared with third parties

1.4 Analytics and Crash Data
We may collect:
device and OS information
app version
anonymized usage events
crash and error reports
Collected via Firebase Analytics and Firebase Crashlytics to improve app stability and performance.

4. How We Use Data
We use data to:
authenticate users
provide salary calculation features
store user data and preferences
monitor performance and fix issues
5. Data Sharing
We do not sell personal data.
Data is shared only with:
Google Firebase services required to operate the App
legal authorities when required by law
Firebase processes data in accordance with Google’s Privacy Policy.
6. Data Security
We implement security measures including:
encrypted connections
secure authentication
Firebase Security Rules
limited access to data
7. Account and Data Deletion
Users can:
delete their account directly within the App
request complete deletion of their personal data
After account deletion, associated data will be deleted or anonymized within a reasonable timeframe.
8. Children’s Privacy
The App is not intended for children under 13 years of age.
9. Policy Updates
We may update this Privacy Policy from time to time. The latest version will always be available in the App or via a public link.
